===== QUESTION =====
小孩要跟我投保，要怎麼申請
===== ANSWER =====
由於您未提供目前小孩的投保狀態，請參考以下兩種情況辦理：1. **若小孩原先是依附另一方家長（如父親/母親）投保**：   請原投保單位透過多憑證網路承保作業線上辦理，或填寫「保險對象退保申報表」（L表）辦理退保。接著由您的投保單位透過網路申請或填寫「保險對象投保申報表」（D表）接保。請務必確保投保日期銜接，以避免眷屬身分重複或中斷。2. **若小孩欲依附祖父母投保**：   原則上小孩應依附父母投保。若有特殊原因無法依附父母，需檢附聲明書說明原因，方可依附祖父母之被保險人以眷屬身分加保。[END]

4cdc8866-987d-40af-b170-bdb0e6b18a96
===== QUESTION =====
公司老闆要怎麼加健保
===== ANSWER =====
原則上，公司負責人必須在該單位以負責人身分辦理加保。不過，若您符合以下特定情況，則不需要以負責人身分在該公司加保（但加保日期必須與單位核准設立日期一致，否則不符合的部分仍需以負責人身分加保）：1. **在其他公司上班**：若您的主要工作是在另一家公司且以員工身分加保，請提供在保證明等文件，並註明該單位的名稱代號。2. **擔任多家公司負責人**：若您同時是兩家以上公司的負責人，且已經在其中一家單位加保，只需註明該單位的名稱代號即可，無需附證明文件。3. **獨資企業行號**：若您沒有僱用員工，且是以獨立勞動或技藝工作獲取報酬，並已在相關職業工會加保，可提供未聘用員工聲明書、工會會員證以及營利事業登記證影本，由承辦人員審核。此外，若公司剛成立且僅有負責人一人，仍應向本署申請成立投保單位並辦理負責人加保；但如果您同時有兩份工作且已在主要工作單位投保，則不需要在新成立的公司重複投保。[END]

===== QUESTION =====
原先爸媽都在哥哥的公司加保，但是我這幾年開刀醫藥費支出很高，想說把爸媽的健保加在我這邊，明年繳稅時列舉扣除額可以多一點，順便節稅，這樣我應該怎麼做?
===== ANSWER =====
若您希望將父母的健保由哥哥名下轉至您的名下投保，請依照下列步驟辦理：首先，請您哥哥的投保單位填寫「保險對象退保申報表」（L表）辦理退保。接著，由您的投保單位填寫「保險對象投保申報表」，並將投保日期設定為原單位退保之日。這兩項手續的辦理順序不限，只要日期銜接即可，以避免保險重複或中斷。關於節稅部分，依所得稅法規定，您本人、配偶以及申報的受扶養直系親屬，每人全年繳納的健保費（含一般健保費及補充保費）均可列舉申報保險費扣除額，且不受保險費扣除額上限（2萬4,000元）的限制。[END]

===== QUESTION =====
我好像有欠費，我要怎麼補繳
===== ANSWER =====
您可以透過以下兩種方式查詢並繳納個人欠費：1. **使用全球資訊網**：登入本署全球資訊網的「網路櫃檯」，進入「承保網路櫃檯」→「一般民眾」→「個人健保資料網路服務作業」中的【保險費明細作業查詢與列印/個人未繳保費查詢及繳費】，勾選要預繳的保費後，選擇【網路繳費】，可用信用卡、活期帳戶、行動支付或網路ATM繳款。2. **使用健保快易通APP**：進入【健保櫃檯】→【保險費各項查詢及申請/未繳保費查詢及繳納】，勾選預繳保費後，點選【繳費】，可用信用卡、活期帳戶、行動支付或銀行行動條碼繳款。[END]
